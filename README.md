<h2 style="text-align: center;">ReactNative+Mobx Project</h2>

## Technologies

- Babel V7
- React (V16.5.0) + Mobx (V4.3.1)
- ReactNative V0.57.0
- React-navigation V2.18.0

## Run Server
```bash
react-native start --reset-cache
Or
react-native start
```

## Build
- Build IOS

```bash
react-native run-ios
```

- Build Android

```bash
react-native run-android
```
